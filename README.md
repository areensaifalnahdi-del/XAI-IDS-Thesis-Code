# XAI-IDS-Thesis-Code

This repository contains the implementation code for my bachelor thesis:

**Evaluating Explainable AI for Trustworthy Machine Learning-Based Intrusion Detection: A Study of SHAP, LIME-Style Explanations, and Temporal Validation**

## Project Description

This project implements a machine learning-based Intrusion Detection System using the CIC-IDS2017 dataset and supplementary validation using UNSW-NB15. The framework evaluates detection performance, explainability, temporal validation, explanation stability, calibration, and proxy user-study results.

## Main Components

- Data preprocessing and binary label conversion
- Random Forest model training
- Baseline comparison with Logistic Regression and Decision Tree
- Same-dataset validation
- Cross-dataset temporal validation
- SHAP global explanation
- Permutation-based LIME-style local explanation
- Explanation stability analysis
- Precision–Recall and calibration analysis
- Supplementary validation on UNSW-NB15
